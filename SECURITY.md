# Security policy

## Supported version

The latest release is supported. This is alpha research software and should not be exposed directly to an untrusted network.

## Reporting

Please use GitHub private vulnerability reporting when available. Do not open a public issue containing an exploit, private document, API key, or database.

## Local-data warning

The default server binds to `127.0.0.1`. Changing it to a public interface may expose source text, provenance, generated answers, and memory exports. Add authentication and transport security before any network deployment.

Online generation and embedding providers receive the content sent to their endpoints. Review the provider's privacy terms before processing confidential or copyrighted material.
